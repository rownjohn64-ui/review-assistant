# 📖 О проекте

**Review Assistant** — система автоматической обработки отзывов клиентов.

Пользователь оставляет отзыв через веб-интерфейс, после чего система:

- сохраняет отзыв в базе данных;
- определяет тональность сообщения;
- отправляет уведомление в Telegram;
- генерирует AI-ответ;
- публикует ответ под отзывом;
- меняет статус на обработанный.

# 🚀 Быстрый запуск

## 1. Клонирование проекта

```bash
git clone https://github.com/rownjohn64-ui/review-assistant.git
cd review-assistant
```

## 2. Настройка окружения

Создайте файл `.env`:

```env
DATABASE_URL=postgresql+asyncpg://postgres:postgres@db:5432/reviews

WORKER_API_TOKEN=change-me-super-secret

OPENAI_API_KEY=your_openai_api_key

TELEGRAM_BOT_TOKEN=your_telegram_bot_token

TELEGRAM_USER_CHAT_ID=your_chat_id
```

## 3. Запуск

```bash
docker-compose up --build
```

## 4. Открыть приложение

```text
http://localhost:8001
```

# 🔄 Принцип работы

1. Пользователь оставляет отзыв.
2. Отзыв сохраняется в PostgreSQL.
3. Отзыв получает статус NEW.
4. Worker находит новые отзывы.
5. Определяется тональность.
6. Отправляется Telegram-уведомление.
7. Генерируется AI-ответ.
8. Создаётся комментарий.
9. Отзыв переводится в статус PROCESSED.
