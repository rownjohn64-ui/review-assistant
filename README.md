# Review Assistant 🤖

## Умная система обработки отзывов с AI-ответами и Telegram-уведомлениями

### Возможности
- Приём отзывов через веб-форму
- Автоматическое определение тональности
- Генерация AI-ответов
- Telegram-уведомления
- PostgreSQL
- Docker Compose

## Быстрый запуск

```bash
git clone https://github.com/rownjohn64-ui/review-assistant.git
cd review-assistant
docker-compose up --build
```
